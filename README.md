# Customer Segmentation & Retention Dashboard

An interactive **Customer Segmentation Dashboard** built using **Python, Machine Learning, and Streamlit** to analyze customer purchasing behavior and identify high-value customers.

This project applies **RFM Analysis** and **K-Means Clustering** to segment customers based on their transaction history and visualize insights through an interactive dashboard.

---

## Project Overview

Understanding customer behavior is crucial for businesses to improve retention and increase revenue.

This dashboard analyzes retail transaction data to:

- Identify high-value customers
- Segment customers based on purchasing behavior
- Track revenue trends
- Provide business insights through interactive visualizations

---

## Key Features

- **RFM Analysis**
  - Recency
  - Frequency
  - Monetary Value

- **Customer Segmentation**
  - Machine Learning using **K-Means Clustering**

- **Interactive Dashboard**
  - Filter customers by country
  - Filter customer segments
  - Dynamic charts and metrics

- **Business Insights**
  - Top high-value customers
  - Revenue by country
  - Revenue trend over time
  - Customer segment insights
